02 / Mission flow

From map
to mission.

Each stage added another piece of autonomy until both robots could operate together in the same task.

  1. 01

    Map the warehouse

    Build and save an occupancy map, tune Nav2, and mark keepout zones.

    SLAM
  2. 02
    MOBILE AUTONOMY

    Approach & dock

    Navigate to a rack, close the final alignment loop, and attach it.

    NAV2 + P

    Find & pick

    Estimate package transforms and servo the UR5 into a safe grasp pose.

    VISION
  4. 04
    SYSTEM INTEGRATION

    Deliver & place

    Transport the rack, coordinate the arm, then sort packages autonomously.

04 / Engineering notes

Built around
feedback.

Every useful motion closes a loop: camera observations update object poses, TF2 keeps frames connected, controllers reduce docking error, and Nav2 continuously replans around the warehouse.
